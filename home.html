WhatsApp Widget Test - All Properties Demo

This page demonstrates the WhatsApp widget with all available configuration properties.

Current Configuration:

data-whatsapp-number: "+1234567890"
data-business-name: "Acme Corporation"
data-default-message: "Hello! I'm interested in your services. Can we chat?"
data-theme-color: "#25D366" (WhatsApp Green)
data-position: "bottom-right"
data-welcome-message: "Welcome to Acme Corp! How can our team assist you today?"
data-logo-url: Custom logo for QR code branding

Available Positions:

  • bottom-right (default)
  • bottom-left
  • top-right
  • top-left

Features:

  • ✅ Floating chat button
  • ✅ Customizable business name and colors
  • ✅ Pre-filled WhatsApp messages
  • ✅ QR code generation with custom branding
  • ✅ Loading animations
  • ✅ Responsive design
  • ✅ Custom logo integration
